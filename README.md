# SwiftDay2

## ✅ 実装内容

### 🕒 時計の表示
- 現在時刻（時:分）をリアルタイム表示
- `DateFormatter`を使用して「分まで」の短い形式に
- `Timer.publish(every: 60...)` で**毎分更新**

### 🎨 デザインカスタマイズ
- フォントサイズ：**36pt** に拡大
- フォント種類：**.rounded** で可愛らしく
- 文字色：**ピンク〜パープルのグラデーション**
- 影：`shadow` を付けて立体感を追加

### 🖼 配置
- 時計は**画面上部（花の上）**に表示
- 中央に3Dモデル（`BeautifulRose.usdz`）を配置
